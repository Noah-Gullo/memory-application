# Memory Card
## Link:
https://memory-card-ten-sooty.vercel.app/

## Description:
A memory game where you try to only click on Pokemon that you have not clicked before. After you click on a Pokemon, the order of each card changes. Try to beat your high score! 

## Reflection:
This project was good practice with React and allowed me to better understand how the hook "useEffect()" works. Additionally, it was good practice of a variety of overall web development skills from animation to managing API calls and promises to developing a better UI/UX. I added a dark/light mode toggle in this project for the first time and aimed to focus on creating an responsive, intuitive UI with better UX. Overall a pretty fun project that has strengthened my understanding of React, CSS animations, and overall web development.
